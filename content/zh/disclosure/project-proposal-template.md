> 本模板适用于基金会全部四个方向的项目。每个项目启动前，
> 按此格式完成立项书并在公开文档栏目发布；立项书是后续账目披露与复盘的基准。

## 1. 基本信息

| 字段 | 内容 |
| ---- | ---- |
| 项目编号 | 如 `edu/university-grant-2026` |
| 项目方向 | 人道主义救援 / 寻亲回家 / 教育支持 / 开源科技 |
| 立项日期 | YYYY-MM-DD |
| 执行周期 | 起止时间，或「持续（按年度复核）」 |
| 负责人 | 姓名或化名（对外可联系） |
| 预算总额 | ¥（上限，超支须另行披露说明） |

## 2. 背景与问题

为什么做这件事：要解决的具体问题、现有的空白、为什么是我们来做。

## 3. 目标与产出

- 可检验的目标（数量、范围、时限）；
- 达成与否的判断标准——复盘时逐条对照。

## 4. 受助对象与标准

谁可以获得资助/援助，准入标准是什么，如何核实。
涉及个人的，写明匿名化与隐私保护方式。

## 5. 预算明细

| 科目 | 金额 | 说明 |
| ---- | ---- | ---- |
| （逐项列出） | ¥ | 用途与估算依据 |
| 合计 | ¥ | 不超过预算总额 |

## 6. 执行与拨付

拨付方式（一次性/分期）、拨付条件、经手人与复核人。

## 7. 披露计划

- 支出进入季度账目，附凭证编号；
- 重大节点（拨付、结项、异常）即时披露；
- 结项后 30 天内发布复盘总结。

## 8. 风险与边界

可能出的问题（冒领、执行偏差、隐私泄露等）与应对方式；
明确本项目**不做**什么。
